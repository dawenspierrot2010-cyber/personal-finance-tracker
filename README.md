# Personal Finance Tracker

A console-based finance tracker written in Python.

## Features

- Add income
- Add expenses
- Edit transactions
- Delete transactions
- Search transactions
- Monthly summaries
- Savings goals
- Financial dashboard
- Graphs

## Technologies

- Python
- Matplotlib

1.0 (Released July 2026)
